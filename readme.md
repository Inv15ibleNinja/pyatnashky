прототип игры в "Пятнашки"

Реализуйте прототип игры в "Пятнашки". Возможно несколько типов решения задачи (от простого к более затратному по времени - на усмотрение соискателя):

    теоретически (словами, наглядными схемами, псевдокодом) описать архитектуру приложения и алгоритм работы
    написать код (на любом языке), который позволит эффективно реализовывать алгоритм игры.
    написать готовый прототип, с интерактивным интерфейсом 
    например: консольное приложение, которое выводит в командной строке матрицу игры 4 на 4. Последующий ввод одной из цифр (4, 6, 8, 2) - дает команду передвинуть фишку (влево, вправо, вверх и вниз) соответственно, после чего программа снова выводит в командной строке обновленную матрицу.





    Массив 4х4 заполняем числами из диапазона от 0 до 15, где 0 иллюстрирует пустую ячейку поля.(Все ли варианты разрешимы?)
    В начале игры отрисовываем поле
    Считываем нажатия кнопок на клавиатуре, проверяем возможность* хода, меняем местами значения ячейки с 0 и следующей
        по направлению движения в зависимости от нажатой клавиши.
    Отрисовываем обновленнное поле.
    После хода можно проверять корректность заполнения массива - нижний ряд 1234, над ним 5678 и т.д., когда массив заполнен корректно,
        выводим сообщение о победе, кол-во ходов до победы, опционально- предложение сыграть еще.
    
    *Нельзя сходить:
      вверх, если 0 в нижнем ряду, т.е. i = size -1
      вниз, если 0 в верхнем,
      влево, если 0 в правом,
      вправо, если 0 в левом.
